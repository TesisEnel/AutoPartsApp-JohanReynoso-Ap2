# AutoPartsApp: Ecommerce de Piezas y Servicios Automotrices

##Descripción General

**AutoPartsApp** es una aplicación móvil de comercio electrónico desarrollada con Kotlin y Jetpack Compose. Su objetivo principal es ofrecer una plataforma moderna, fluida y visualmente atractiva para la compra de piezas automotrices y la contratación de servicios de mantenimiento y reparación.

La aplicación está diseñada para manejar un catálogo diverso que incluye soporte para:

* **Autos ligeros**
* **Vehículos pesados**
* **Motores**

El enfoque está en proporcionar una experiencia de usuario simple, con navegación intuitiva, un catálogo completo y soporte robusto para las transacciones y compras en línea.
